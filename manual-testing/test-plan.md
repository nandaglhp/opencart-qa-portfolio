## 1. Overview
| | |
|---|---|
| **Project** | OpenCart E-Commerce Platform |
| **Tester** | Ananda Galih Pratiwi |
| **Version** | OpenCart Demo |
| **Date** | June 2026 |
| **Type** | Manual Functional Testing |

---

## 2. Objectives
- Validate core e-commerce flows work as expected
- Identify functional defects across customer-facing features
- Ensure critical user journeys (register, search, checkout) are bug-free

---

## 3. Scope

### In Scope
- User Registration & Login
- Product Search & Filter
- Product Detail Page
- Shopping Cart
- Checkout Flow
- User Account Management

### Out of Scope
- Admin panel (Phase 1)
- Payment gateway integration
- Performance & security testing (later phases)

---

## 4. Test Approach
- Black-box functional testing
- Equivalence partitioning & boundary value analysis for input fields
- Exploratory testing for edge cases
- Test cases documented in Markdown (GitHub)

---

## 5. Entry & Exit Criteria

### Entry Criteria
- Application is accessible and stable
- Test cases are prepared

### Exit Criteria
- All planned test cases executed
- All critical/high bugs reported
- Test summary report completed

---
